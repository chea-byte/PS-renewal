📥 How to export from OfficeRND

Do this once a week before your renewal review. Takes under 2 minutes.

  1. Log in to OfficeRND → go to Reports in the left menu
  2. Select Memberships report
  3. Set filter: Plan = Private Suite (PS) · Status = Active
  4. Click Export → CSV (top right)
  5. Open the downloaded CSV in any text editor (Notepad, TextEdit)
  6. Select all (Ctrl+A) → Copy (Ctrl+C)
  7. Come back here → click Paste CSV instead → below → paste → Load
Required columns in the export: company_name · outlet_code · room_name · current_pax_count · membership_start_date · membership_end_date · current_monthly_rate · deposit_held · plan

The Google Sheet is your accountability system — it tracks whether actions have been taken. The Pricing Tool is your calculation engine — it tells you what to offer and when to send the notice. Use both; they do different jobs.

Common questions

Do I need to publish the Google Sheet for the Tool to load it?
No. Set sharing to "Anyone with the link can view" and paste the URL in Settings. If that fails (common when running locally), use Settings → Paste CSV instead.
The tool shows fewer companies than the Sheet. Why?
The Tool shows PS (Private Suite) contracts only. VOs, HDs, and other plans are excluded. Set Window to "All PS" to see all PS contracts regardless of renewal date.
Can I trust the tool's pricing to send to clients?
Use it as a starting point. The draft email is a template — always review before sending. For Key Accounts and Promo Rate companies, AM approval is required before any offer goes out.
The clause / notice date doesn't match the signed MA. Which is right?
Always trust the signed MA. The tool calculates from OfficeRND dates — if the start date in ORND is wrong, the clause calculation will be off. Verify the contract for any dispute.
